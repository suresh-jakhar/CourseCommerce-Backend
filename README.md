# CourseCommons

CourseCommons is an open learning platform where **teachers can create courses and students can enroll to learn**, while also enabling a **community driven learning ecosystem**.

The vision behind CourseCommons is to build a space where knowledge is **shared, structured, and accessible**, allowing educators to publish courses and learners to engage with content and interact with the community.

---

## 🚀 Current Status

⚠️ This project is **actively under development**.

At the moment, only the **core backend foundation** has been implemented. Many planned features such as community interaction, announcements, and advanced course management are still in progress.

---

## ✨ Planned Vision

CourseCommons aims to become a **complete learning ecosystem**, including:

* Teachers creating and managing courses
* Students enrolling in courses
* Free and paid course support
* Community interaction between students and teachers
* Teacher channels for announcements
* Discussion threads
* Learning communities around courses
